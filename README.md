## Git Commands Used in This Assignment

### Repository Setup
- `git clone [url]`
- `git config --global user.name`
- `git config --global user.email`

### Basic Operations
- `git status`
- `git add`
- `git commit -m`
- `git push`
- `git pull`

### Branching & Merging
- `git branch`
- `git checkout -b`
- `git checkout`
- `git merge`

### Conflict Resolution
- Manually edited the conflicted file to remove conflict markers
- `git add` (mark conflict resolved)
- `git commit` (complete merge)

### History / Inspection
- `git log --oneline --graph`
- `git diff`
